# Covid API

1. Clone project
2. `npm install`
3. Clone `.env.template` and rename to `.env.template`
4. Change environment variables
5. Run database

```
docker-compose up -d
```

6. Run app

```
npm run start:dev
```

7. Execute SEED

```
http://localhost:3000/api/seed
```
